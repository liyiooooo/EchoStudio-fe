# EchoStudio音樂網站專案
## 負責部分 
* ui設計：主頁（初稿）、會員登入、註冊、忘記密碼 、會員主頁、會員優惠券、會員收藏
* 切版/功能：會員訂單、會員訂單詳細頁、會員優惠券、會員課程、訂單評論

---
## 使用技術
* 前端：React、HTML5、CSS3、JavaScript、SCSS
* 後端：Node.js
---
